### The Project

Users should be able to:
   
- Saves name in local storage
- background images (random) using API

- Pomodoro system
- Count down (Add, Edit, Delete)
- Weather and location using API
- Habit Tracker
- Quotes (random)
- To do list (Add, Edit, Delete)

- Time and greeting text
